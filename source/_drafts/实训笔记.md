作用域

```javascript
var b = 100;

function fun1(){
    console.log("全局变量 B" + b);
}

fun1()
```

1. 在函数内部创建一个没有var的变量，就自动变成全局变量
2. 关于作用域上升的问题，我们使用块级作用域
3. 不需要变的都用常量const，还能提高运行性能
4. ES6允许用箭头定义函数，就跟python中的lambda的那个匿名函数一样
